# VS Code Devcontainer

Среда разработки (Development Container) на базе Docker, поддерживаемая редактором Visual Studio Code. Официальный репозиторий `home-assistant/core` включает преднастроенный devcontainer, который содержит все необходимые зависимости для запуска, тестирования и отладки компонентов ([[Debugging Custom Components]]) в изоляции от реальной инсталляции умного дома.
