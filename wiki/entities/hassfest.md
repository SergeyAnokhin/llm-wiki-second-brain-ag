# Hassfest

Официальная утилита валидации для компонентов Home Assistant. Проверяет корректность файла `manifest.json`, структуру директорий и другие стандарты платформы. Часто интегрируется в [[Continuous Integration (CI)]] процессы через [[GitHub Actions]].
