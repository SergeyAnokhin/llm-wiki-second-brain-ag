# GitHub API

Внешний API, используемый в качестве примера источника данных при обучении созданию пользовательского компонента ([[Home Assistant Custom Component]]). В современных интеграциях вызов подобных API должен осуществляться с использованием асинхронных подходов ([[Asynchronous Integration Setup]]).
