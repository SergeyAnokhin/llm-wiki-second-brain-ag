# pytest-home-assistant

(Официальное название `pytest-homeassistant-custom-component`) — пакет фикстур (fixtures) для библиотеки pytest. Значительно упрощает написание unit-тестов для кастомных компонентов, позволяя эмулировать вызовы системных функций (например, флоу конфигурации) без необходимости запускать полноценный инстанс Home Assistant.
