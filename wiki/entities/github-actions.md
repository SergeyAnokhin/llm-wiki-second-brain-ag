# GitHub Actions

Инструмент автоматизации рабочих процессов, встроенный в GitHub. Широко используется для [[Continuous Integration (CI)]] в разработке [[Home Assistant Custom Component]], позволяя автоматически запускать тесты (`pytest`) и проверки ([[Hassfest]]) при каждом push или pull request.
