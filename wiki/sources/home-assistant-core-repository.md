# Home Assistant Core Repository

**Источник:** https://github.com/home-assistant/core
**Тип:** Официальный исходный код ядра и встроенных компонентов

## Краткое содержание
Огромный репозиторий, содержащий само ядро Home Assistant и все официально поддерживаемые встроенные компоненты (директория `homeassistant/components/`).

## Для чего нужен этот проект?
- Это "золотой стандарт" и энциклопедия лучших практик.
- Любой разработчик пользовательских компонентов (custom components) должен обращаться сюда, чтобы смотреть, как реализованы "платиновые" (platinum quality) интеграции.
- Здесь можно подсмотреть недокументированные возможности API, правильное использование типов, работу `pre-commit` хуков и структуру автоматического тестирования (`pytest`).

## Интересные находки (что искать)
- Папка `tests/` — гигантская база примеров для написания unit-тестов.
- Папка `homeassistant/components/` — эталонные реализации (например, компонент `mqtt` или `http`), где можно посмотреть, как правильно использовать `async_setup_entry`.
- Файлы настроек `lint` и форматирования (например, `pylintrc`, `black`, `isort`), которые стоит перенести в свой проект для соблюдения стандартов.
