# System Info

**Source:** System info.md
**Date ingested:** 2026-05-09
**Type:** notes

## Summary

Этот текст представляет собой технический паспорт сетевого хранилища [[synology-ds212j|Synology DS212j]], отображающий основные параметры работы системы. Документ содержит сведения о модели процессора, объеме оперативной памяти, версии программной среды и времени непрерывной работы (uptime). Эта информация служит инструментом мониторинга и диагностики статуса сервера.

## Key Claims

- Сетевое хранилище (NAS) идентифицируется как модель [[synology-ds212j|Synology DS212j]] от компании [[synology|Synology]].
- Устройство работает на базе одноядерного процессора [[marvell-kirkwood-88f6281|MARVELL Kirkwood 88F6281]] с тактовой частотой 1.2 GHz.
- Объем физической памяти составляет 256 МБ.
- В качестве операционной системы используется [[dsm|DSM]] (DiskStation Manager) версии 6.1.7-15284 Update 3.
- Система синхронизирует системное время через сервер `time.google.com`.

## Entities Mentioned

- [[synology|Synology]] — Производитель оборудования, в частности сетевых хранилищ.
- [[synology-ds212j|Synology DS212j]] — Конкретная модель NAS из паспорта.
- [[marvell-kirkwood-88f6281|MARVELL Kirkwood 88F6281]] — Процессор, установленный в устройстве.

## Concepts Covered

- [[dsm|DSM]] — Операционная система, используемая на устройствах Synology.
