# NFS

Network File System — сетевой протокол без сохранения состояния (stateless). Благодаря отсутствию постоянной сессии, NFS оказывает минимальное влияние на фоновую дисковую активность и рекомендуется для использования совместно со спящим режимом (Hibernation).
