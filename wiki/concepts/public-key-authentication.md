# Public Key Authentication

Метод аутентификации в сети (например, для протокола SSH), использующий пару ключей — открытый и закрытый. Открытый ключ размещается на сервере, а закрытый остается у клиента, что позволяет устанавливать защищенное соединение без ввода пароля.
