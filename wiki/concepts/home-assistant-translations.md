# Home Assistant Translations

Система локализации, используемая в пользовательских компонентах ([[Home Assistant Custom Component]]) для отображения текста, названий полей, сообщений об ошибках и статусов в UI на нужном языке (особенно важно для [[Config Flow]]). Файлы переводов размещаются в папке `translations/` и называются по ISO 639-2 коду языка (например, `en.json`, `ru.json`).
